# ZeroQ Umbrel Apps

Custom Umbrel app store with curated applications.

## Apps

- **Riven** - Plex/Jellyfin torrent streaming through Debrid services (Real-Debrid, All-Debrid)

## Installation

Add this app store to Umbrel:

1. Open your Umbrel dashboard
2. Go to App Store
3. Click "Community App Stores"
4. Add: `https://github.com/ZeroQ-bit/My-Umbrel-Apps`
